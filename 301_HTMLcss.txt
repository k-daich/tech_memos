---------------------------------------------- セレクタ ------------------------------------------------
■A以下のすべてのB(子孫セレクタ)
	A B

■A直下のB(子セレクタ)
	A > B

■A直後のB(隣接セレクタ)
	A + B

■Aと同階層でA以降にあるB(間接セレクタ)
	A ~ B


---------------------------------------------- text ------------------------------------------------
■文字に関するスタイル指定
	color: ***; 文字色を指定する
	font-size: ***; 文字サイズを指定する
	font-weight: ***; 文字の太さを指定する
	font-style: ***; イタリック体にする
	font-family: ***; フォントを指定する
	font-variant: small-caps; スモールキャピタルを指定する
	font: ***; フォントの設定をひとまとめに行う

■remの使用例
	html {
	  font-size: 15px;
	}

	p {
	  font-size: 1.2rem;
	}

	strong {
	  font-size: 1.4rem;
	}
